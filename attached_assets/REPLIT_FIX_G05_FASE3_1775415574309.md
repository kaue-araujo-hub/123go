## O PROBLEMA (conforme imagem capturada)

A **fase 3 do jogo Rã Puladora está injogável**.

O que está acontecendo:
- A fase 3 está gerando **quantidades iguais nas duas lagoas** (ex: 12 vs 12)
- Quando isso acontece, o jogo exibe a mensagem "As lagoas têm a MESMA quantidade!"
- Aparece apenas o botão **"São Iguais! ✅"** como única opção
- A criança só precisa clicar num botão — **sem swipe, sem escolha, sem jogo**
- A mecânica central do jogo (deslizar a rã para a lagoa correta) **não acontece**

O que deveria acontecer na fase 3:
- As duas lagoas têm **quantidades DIFERENTES**
- A criança deve **deslizar a rã** para a lagoa com a quantidade correta
- conforme a pergunta exibida ("mais flores" ou "menos flores")
- A diferença entre as quantidades deve ser **pequena** (1 a 3 flores de diferença)
  para aumentar a dificuldade em relação às fases anteriores

---

## CAUSA RAIZ DO BUG

O gerador de quantidades da fase 3 **não está garantindo que os dois valores sejam
diferentes**. Provavelmente o código usa algo como:

```js
// CÓDIGO BUGADO — pode gerar valores iguais:
const qty1 = randomInRange(difficulty)  // ex: retorna 12
const qty2 = randomInRange(difficulty)  // ex: também retorna 12 — PROBLEMA
```

Quando `qty1 === qty2`, o jogo entra no caminho de código do "São Iguais" e
exibe o botão estático em vez da mecânica de swipe.

### PASSO 2 — Corrija a função geradora de quantidades

Localize a função que gera as quantidades das lagoas
(pode se chamar `generateLagoas`, `generateRound`, `getFlowerCounts`, ou similar)
e substitua completamente por esta versão:

```js
/**
 * Gera as quantidades de flores das lagoas para uma rodada.
 * GARANTE que as quantidades são sempre diferentes (allowEqual deve ser false na fase 3).
 *
 * @param {object} phaseConfig — configuração da fase atual do array PHASES
 * @returns {{ qty1: number, qty2: number, correct: 'left'|'right', question: 'mais'|'menos' }}
 */
function generateRound(phaseConfig) {
  const { min, max, minDiff = 1, maxDiff = 5, allowEqual = false } = phaseConfig

  let qty1, qty2
  let attempts = 0

  do {
    qty1 = min + Math.floor(Math.random() * (max - min + 1))
    qty2 = min + Math.floor(Math.random() * (max - min + 1))
    attempts++

    // Proteção contra loop infinito (máx 100 tentativas)
    if (attempts > 100) {
      // Força diferença manualmente como fallback seguro
      qty1 = min + Math.floor((max - min) * 0.4)
      qty2 = qty1 + minDiff + Math.floor(Math.random() * (maxDiff - minDiff + 1))
      break
    }
  } while (
    // Repetir enquanto qualquer condição inválida for verdadeira:
    qty1 === qty2 ||                                   // ← nunca iguais (a não ser que allowEqual)
    (!allowEqual && qty1 === qty2) ||                  // ← proteção dupla
    Math.abs(qty1 - qty2) < minDiff ||                 // ← diferença mínima não atingida
    Math.abs(qty1 - qty2) > maxDiff                    // ← diferença máxima não excedida
  )

  // Define aleatoriamente a pergunta (mais ou menos)
  const question = Math.random() > 0.5 ? 'mais' : 'menos'

  // Define qual lagoa é a resposta correta
  let correct
  if (question === 'mais') {
    correct = qty1 > qty2 ? 'left' : 'right'
  } else {
    correct = qty1 < qty2 ? 'left' : 'right'
  }

  return { qty1, qty2, correct, question }
}
```
