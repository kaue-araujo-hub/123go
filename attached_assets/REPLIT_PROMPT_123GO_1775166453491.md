# PROMPT PARA O REPLIT — PROJETO 123GO!

---

## CONTEXTO DO PROJETO

Crie um projeto web completo chamado **123GO!** — uma plataforma educacional de jogos digitais de matemática para crianças do 1º ao 3º ano do Ensino Fundamental, alinhada ao **Currículo Paulista** (São Paulo, Brasil). O projeto deve ser uma Single Page Application (SPA) funcional, com página de catálogo de jogos e os 21 jogos interativos implementados.

---

## STACK TECNOLÓGICA E DEPENDÊNCIAS

### Linguagem e Runtime
- **Node.js** (v20 LTS) como runtime para o servidor de desenvolvimento
- **Vite** como bundler e dev server (instalação: `npm create vite@latest 123go -- --template vanilla`)

### Dependências principais (`npm install`)
```
howler          # áudio: sons de acerto, erro, feedback musical por fase
gsap            # animações avançadas: bounce, shake, grow/shrink, color change
@dnd-kit/core   # drag-and-drop acessível para touch e mouse
@dnd-kit/sortable
@dnd-kit/utilities
canvas-confetti # efeito de confete ao completar fases
tone            # síntese musical procedural (música que evolui com progresso)
```

### Dependências de dev (`npm install -D`)
```
vite
@vitejs/plugin-legacy   # compatibilidade com browsers antigos em tablets escolares
vite-plugin-pwa         # PWA para funcionar offline em escolas sem internet
workbox-window
```

### Fontes (via Google Fonts CDN no HTML)

## OS 21 JOGOS — ESPECIFICAÇÃO PEDAGÓGICA E DE INTERAÇÃO

### Regras gerais de todos os jogos

1. **5 fases progressivas** por jogo — dificuldade cresce gradualmente
2. **Consistência de interação**: se a fase 1 usa arrastar, todas as 5 fases usam arrastar (o tipo de interação principal não muda entre fases)
3. **Feedback sonoro obrigatório** via Howler.js:
   - Acerto: som "ding" / "pop" alegre
   - Erro: som suave de "womp" (não punitivo)
   - Fase completa: jingle de celebração
   - Jogo completo: música festiva + confete (canvas-confetti)
4. **Feedback visual via GSAP**:
   - Acerto: elemento correto faz `scale(1→1.2→1)` + muda para verde momentaneamente
   - Erro: elemento errado faz `shake` lateral (3x rápido) + fica vermelho por 0.5s
   - Fase completa: estrelinhas voam pela tela
5. **Música progressiva via Tone.js**: BPM e complexidade harmônica aumentam a cada fase
6. **Touch-friendly**: todos os alvos de toque ≥ 44x44px (WCAG 2.5.5)
7. **Barra de progresso** visual no topo de cada jogo (fase atual / 5)
8. **Botão voltar** para o catálogo em todos os jogos (top-left)

### JOGO 1 — Festa da Lagarta
**Unidade:** Números | **Habilidade:** EF01MA02 | **Interação principal:** Arrastar e soltar
- Cena: lagarta animada em fundo verde floresta
- Mechânica: folhas espalhadas pela tela; criança arrasta para a boca da lagarta
- A cada folha correta: lagarta cresce visualmente (GSAP scaleX)
- Fase 1: contar até 5 folhas | Fase 2: até 10 | Fase 3: pareamento lagarta↔folhas | Fase 4: velocidade aumenta | Fase 5: agrupamento de 2 em 2
- Sons: "munch" ao engolir folha, barriga inchando ao errar

### JOGO 2 — Par ou Ímpar?
**Unidade:** Números | **Habilidade:** EF01MA02 | **Interação principal:** Toque/clique
- Mechânica: objetos aparecem em pares na tela; criança toca no que ficou sozinho
- Troca de tema a cada fase (meias → luvas → sapatos → planetas → frutas)
- Fase 1: 4 elementos | Fase 2: 6 elementos | Fase 3: pares escondidos atrás de nuvens | Fase 4: toque no ritmo da música | Fase 5: coleção misturada com animação surpresa ao concluir
- Sons: "clique mágico" ao selecionar, fanfarra ao acertar o par

### JOGO 3 — Caça Estrelas
**Unidade:** Números | **Habilidade:** EF01MA02 | **Interação principal:** Toque temporizado
- Mechânica: flash visual de constelação por 2s, depois some; criança escolhe entre 3 números
- A estrela correta pisca no ritmo da música (usar Tone.js para sincronismo)
- Fase 1: flash 2s, até 5 estrelas | Fase 2: flash 1.5s, até 10 | Fase 3: agrupadas em constelações | Fase 4: comparação de 2 céus | Fase 5: ordenar 3 céus do menor para o maior
- Efeito: céu animado com estrelas piscando em parallax

### JOGO 4 — Loja de Balas
**Unidade:** Números | **Habilidade:** EF01MA03 | **Interação principal:** Toque/clique
- Cena: balconista animado atrás do balcão, dois potes transparentes com balas coloridas
- Mechânica: toque no pote com mais/menos balas conforme a pergunta
- Fase 1: diferença grande (5 vs 15) | Fase 2: diferença média (10 vs 14) | Fase 3: igualar arrastando balas | Fase 4: três potes ordenados | Fase 5: flash rápido + escolha antes do tempo
- Sons: "tchim-tchim" ao acertar, balconista dança (GSAP)

### JOGO 5 — Rã Puladora
**Unidade:** Números | **Habilidade:** EF01MA03 | **Interação principal:** Deslizar (swipe)
- Cena: duas lagoas com flores flutuantes
- Mechânica: deslize a rã na direção da lagoa correta
- Deslize em arco = salto animado (GSAP bezier path)
- Fase 1: 20 vs 10 | Fase 2: 15 vs 18 | Fase 3: quantidade igual (rã não salta — curiosidade) | Fase 4: três lagoas | Fase 5: corrida contra tempo com 5 comparações
- Sons: "splash" ao cair na lagoa, coaxar ao acertar

### JOGO 6 — Balões da Festa
**Unidade:** Números | **Habilidade:** EF01MA03 | **Interação principal:** Toque (estouro)
- Cena: balões coloridos subindo em dois grupos
- Mechânica: toque nos balões para estourá-los e igualar os grupos
- Fase 1: 20 vs 12, estoure 8 | Fase 2: 15 vs 22 | Fase 3: comparação entre 3 grupos | Fase 4: balões sobem e somem — velocidade | Fase 5: criar os dois grupos iguais do zero
- Sons: "pop" animado com partículas coloridas ao estourar (canvas-confetti localizado)

### JOGO 7 — Trem dos Números
**Unidade:** Números | **Habilidade:** EF01MA06 | **Interação principal:** Arrastar e soltar
- Cena: trem em movimento com vagões numerados, um vagão com lacuna
- Mechânica: arraste o número correto para o vagão vazio
- Música de trem que acelera conforme acertos (Tone.js BPM++)
- Fase 1: adição até 5 | Fase 2: adição até 10 | Fase 3: subtração até 5 | Fase 4: subtração até 10 | Fase 5: mix adição+subtração com contador de tempo
- Sons: apito de trem, "clonk" metálico ao encaixar

### JOGO 8 — Pizzaria Mágica
**Unidade:** Números | **Habilidade:** EF01MA06 | **Interação principal:** Arrastar e soltar
- Cena: balcão de pizzaria com prato e fatias
- Mechânica: arraste fatias para completar o pedido do cliente
- Temas mudam por fase: pizza → sanduíche → sorvete → brigadeiro → bolo
- Fase 1: complemento até 5 | Fase 2: até 10 | Fase 3: subtração (cliente devolveu) | Fase 4: soma de dois pratos | Fase 5: monte você mesmo a pizza com X fatias
- Sons: "ding" de caixa registradora ao completar pedido

### JOGO 9 — Batalha de Constelações
**Unidade:** Números | **Habilidade:** EF01MA06 | **Interação principal:** Gesto de desenho (touch path)
- Cena: céu noturno com estrelas
- Mechânica: desenhe o símbolo + ou – com o dedo na tela para resolver a conta

## MOTOR DE JOGO — GameEngine.js

Todas as telas de jogo herdam da classe base `GameEngine`:

```js
class GameEngine {
  constructor(config) {
    this.gameId = config.id
    this.totalPhases = 5
    this.currentPhase = 1
    this.score = 0
    this.audio = new AudioManager()
    this.phases = new PhaseManager(this.totalPhases)
    this.feedback = new FeedbackSystem()
  }

  onCorrect(element) {
    this.score++
    this.feedback.correct(element)      // GSAP bounce + som ding
    this.audio.playCorrect()
  }

  onWrong(element) {
    this.feedback.wrong(element)        // GSAP shake + som womp
    this.audio.playWrong()
  }

  advancePhase() {
    this.currentPhase++
    if (this.currentPhase > this.totalPhases) {
      this.gameComplete()
    } else {
      this.phases.advance(this.currentPhase)
      this.audio.advanceMusic(this.currentPhase)  // Tone.js aumenta BPM
    }
  }

  gameComplete() {
    confetti({ particleCount: 150, spread: 70, origin: { y: 0.6 } })
    this.audio.playVictory()
  }
}
```
